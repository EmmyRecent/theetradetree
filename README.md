# Theetradetree

**Theetradetree** is an ecommerce platform that aims to connect vendors with customers.

## 🛠️ Tech Stack

- **Frontend**: React Native, TypeScript
- **Backend**: Firebase
- **Styling**: NativeWind 
- **Version Control**: Git & GitHub

---

## 🚀 Getting Started

### 📥 Installation

```bash
# Clone the repo
git clone https://github.com/EmmyRecent/theetradetree.git

# Navigate into the project
cd theetradetree

# Install dependencies
npm install

# Run locally
npm run dev
```
